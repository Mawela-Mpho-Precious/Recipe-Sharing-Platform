# Recipe Sharing Platform üç≤

A Django-based web application for sharing, browsing, and managing recipes. Users can register, submit recipes, search recipes, view categories, and explore featured meals.

---



## Features
- User registration and login system
- Submit, edit, and delete recipes
- Browse recipes by categories
- Favorites and collections
- Admin dashboard for managing recipes
- Media uploads for recipe images

---

## Tech Stack
- **Backend:** Django (Python)
- **Frontend:** HTML, CSS, Django Templates
- **Database:** SQLite (development)
- **Version Control:** Git & GitHub
